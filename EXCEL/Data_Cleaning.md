# Data Cleaning in Excel

## 🎯 Qëllimi
Pastrimi i të dhënave (kritike në banka).

---

## ✂️ TRIM
```excel
=TRIM(A2)
```

---

## 🧼 CLEAN
```excel
=CLEAN(A2)
```

---

## 🔠 UPPER
```excel
=UPPER(A2)
```

---

## 🔡 LOWER
```excel
=LOWER(A2)
```

---

## 🧠 PROPER
```excel
=PROPER(A2)
```

---

## 🔁 SUBSTITUTE
```excel
SUBSTITUTE(text, old_text, new_text)
```

```excel
=SUBSTITUTE(A2,"-","")
```

---

## 🔍 FIND
```excel
=FIND("a",A2)
```

---

## 🔍 SEARCH
```excel
=SEARCH("a",A2)
```

---

## 🔗 CONCAT / CONCATENATE
```excel
=CONCAT(A2,B2)
```

```excel
=A2 & " " & B2
```

---

## ✂️ LEFT
```excel
=LEFT(A2,5)
```

---

## ✂️ RIGHT
```excel
=RIGHT(A2,3)
```

---

## ✂️ MID
```excel
=MID(A2,2,5)
```

---

## 🔢 VALUE
```excel
=VALUE(A2)
```

---

## 💡 Use Case

- pastrim emrash klientësh
- standardizim ID
- formatim adresash
- import nga sisteme të ndryshme
